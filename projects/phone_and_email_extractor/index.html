Project Overview:

Imagine you could quickly find email addresses and phone numbers in large bodies of texts by simply pressing CTRL + A to select all the text, CTRL + C to copy it to the clipboard, and through this widget that we're going to build, pressing CTRL + V in order to paste only the desired text, in this case, phone numbers and emails addresses.

Well imagine no longer, lets get to building one.

TAKE NOTE:

  1. Get the text off the clipboard.
  2. Find all phone numbers and email addresses in the text.
  3. Paste them onto the clipboard.

CODE REQUIREMENTS:

  1. Use the pyperclip module to copy and paste strings.
  2. Create two regexes, one for matching phone numbers the other for matching email addresses.
  3. Find all matches, not just the first match, of both regexes.
  4. Neatly format the matched strings into a single string to paste.
  5. Display some kind of message if no matches were found in the text.
